# Nextui-Vite

A vite typescript app to test [nextui](https://nextui.org/) for production

## Getting started
clone the app

```bash
git clone https://github.com/PatrickNiyogitare28/nextui-vite.git
```

```bash
cd nextui-vite
```

```bash
yarn install
```

### Running the dev server
```bash
yarn dev
```

### Building for production
```bash
yarn build
```
